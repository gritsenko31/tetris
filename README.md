## ğŸ® How to Play

### Objective
Stack falling tetrominoes to create complete horizontal lines. Clear lines to score points and prevent blocks from reaching the top!

### Game Rules

- Tetrominoes fall from top of the screen
- Move and rotate pieces before they land
- Complete horizontal lines to clear them
- Cleared lines disappear and blocks above fall down
- Game ends when blocks reach the top
- Speed increases every level
